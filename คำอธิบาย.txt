3SB04:Framework Django
เป็นโปรแกรม Opensource สำหรับพัฒนาเว็บแอปพลิเคชัน เขียนด้วยภาษา Python   
โดยเป้าหมายหลักในการทำให้เว็บไซต์ทำการเชื่อมต่อกับดาต้าเบส ทำให้ลดความซับซ้อนในส่วนนั้นลง
ข้อดี : ระบบจัดการ DATABASE ที่ใช้งานง่าย มีในส่วนของระบบ Admin สำเร็จรูปมาให้ใช้ , Plugins เยอะมากเพราะมีการเขียนด้วยภาษา Python ทำให้ package เยอะ
ข้อเสีย : Host ในไทยไม่รองรับการรัน ต้องมีการใช้งานจาก VPS,Service รูปแบบอื่น , การอัปโหลดในส่วนของ Server ค่อนข้างยาก
